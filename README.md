# Adaptive Image Enhancement

Веб-приложение для автоматической коррекции изображений в браузере пользователя.  
Система анализирует фотографию локально, подбирает коэффициенты улучшения и применяет коррекцию без отправки изображения на сервер.

## Назначение

Проект предназначен для улучшения качества изображений по трём основным параметрам:

- яркость;
- контрастность;
- цветность.

Приложение работает полностью на стороне клиента. Пользователь загружает изображение, запускает обработку, отслеживает прогресс выполнения и получает готовый результат для скачивания.

## Возможности

В текущей версии доступны:

- загрузка изображений через веб-интерфейс;
- создание задачи на обработку;
- отображение статуса и прогресса;
- отмена активной задачи;
- просмотр исходного изображения;
- просмотр обработанного результата;
- скачивание итогового файла;
- выполнение обработки в фоне без блокировки интерфейса;
- поддержка форматов JPG, JPEG, PNG, BMP, HEIC и HEIF.

## Принцип работы

Обработка изображения выполняется по следующему сценарию:

1. Пользователь выбирает файл в интерфейсе.
2. Приложение создаёт задачу и присваивает ей идентификатор.
3. Исходное изображение передаётся в `Web Worker`.
4. На этапе декодирования:
   - стандартные форматы открываются напрямую средствами браузера;
   - HEIC и HEIF при необходимости предварительно конвертируются в JPEG.
5. Из изображения формируется уменьшенная копия фиксированного размера для подачи в модель.
6. ML-модель вычисляет коэффициенты коррекции:
   - яркости;
   - контрастности;
   - цветности.
7. Полученные коэффициенты применяются к исходному изображению в полном разрешении.
8. Результат кодируется в выходной формат и возвращается в интерфейс.
9. Пользователь получает готовое изображение и может сохранить его локально.

## Подход к ML-части

В проекте используется подход регрессии глобальных параметров изображения.  
Модель предсказывает не само улучшенное изображение, а набор коэффициентов, которые затем применяются отдельным алгоритмом постобработки.

Такой подход выбран по нескольким причинам:

- компактный размер модели;
- более низкая вычислительная нагрузка;
- быстрый запуск в браузере;
- удобная интеграция с клиентским приложением.

## Подготовка модели

Для обучения использовались эталонные изображения и их синтетически ухудшенные версии.  
На этапе подготовки данных варьировались параметры яркости, контрастности и цветности, после чего рассчитывались коэффициенты восстановления.

В результате модель обучалась предсказывать вектор из трёх значений:

- коэффициент коррекции яркости;
- коэффициент коррекции контрастности;
- коэффициент коррекции цветности.

Обучение выполнялось во внешней среде на Python с использованием TensorFlow и Keras, после чего модель была экспортирована для браузерного инференса через TensorFlow.js.

## Архитектура

Проект разделён на несколько логических слоёв.

### Интерфейс

Интерфейс отвечает за:

- загрузку изображений;
- отображение задач;
- отображение прогресса;
- предпросмотр исходного и обработанного изображения;
- скачивание результата.

### Управление задачами

Слой управления задачами отвечает за:

- создание задачи;
- хранение текущего состояния;
- обновление прогресса;
- отмену обработки;
- передачу событий в UI.

### Worker-пайплайн

Worker-слой отвечает за:

- декодирование изображения;
- подготовку входа для модели;
- запуск инференса;
- применение коэффициентов коррекции;
- экспорт результата.

### ML-слой

ML-слой отвечает за:

- загрузку модели;
- подготовку тензора;
- запуск предсказания;
- нормализацию итоговых коэффициентов.

## Установка и запуск

Требования:

- Node.js
- npm

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки:

```bash
npm run dev
```

Сборка production-версии:

```bash
npm run build
```

Локальный просмотр production-сборки:

```bash
npm run preview
```

## Ограничения

Текущая версия ориентирована на запуск в современных браузерах с поддержкой необходимых Web API.  
Скорость обработки зависит от разрешения изображения, ресурсов устройства и поддержки аппаратного ускорения в браузере.
